## 👩‍💻 About Me

I build **distributed backend systems** and **platform infrastructure**.

Currently at **Advance Auto Parts**, working on **Commerce Services** — building and operating large-scale microservices using:

- Java &amp; Spring Boot  
- Apache Kafka  
- Kubernetes &amp; OpenShift  

I focus on **scalability, resilience, and observability** in production systems.

Outside work, I’ve built across the platform stack:

- 📊 Observability → Prometheus, Grafana, Loki, Jaeger  
- ⚙️ Platform Engineering → Kubernetes, Helm, ArgoCD  
- ☁️ Infrastructure → Terraform on AWS  

📄 **IEEE Published Researcher (ICSCNA 2023)**
